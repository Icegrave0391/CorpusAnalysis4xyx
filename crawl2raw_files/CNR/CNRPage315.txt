>>>各地采取周密精准、有效措施 防止新型冠状病毒感染肺炎疫情蔓延

央广网北京2月6日消息 据中央广播电视总台中国之声《新闻和报纸摘要》报道，目前，各地正在采取更加周密精准、管用有效的措施，防止新型冠状病毒感染肺炎疫情蔓延。2020-02-06 新闻和报纸摘要全文&gt;&gt;&gt;
武汉是新型冠状病毒感染肺炎疫情的重点地区。在武汉，社区是疫情防控的“第一线”，也是疫情严守的最后一道防线。武汉西马街道香港社区书记张静说，党员联合群众、物业公司的力量，筑起了一道防疫堡垒。
张静：更多的时候是依靠群众、依靠我们的党员同志，依靠我们的门栋组长，如果说有发热病人，我们会在他家门口，包括楼道全部进行消杀。如果没有发热病人门栋的话，我们就会在楼道，花坛、垃圾桶进行消杀。
在病毒的检测能力方面，国家呼吸系统疾病临床医学研究中心武汉病毒诊断研究分中心近日（4日）正式挂牌成立，将协助武汉汉口医院、金银潭医院、协和医院进行病毒核酸检测工作，把单日检测能力逐步从1000例提升到5000例以上。
为进一步减少人员流动，浙江杭州对全市所有村庄、小区、单位实行封闭式管理。温州、台州、衢州、金华、绍兴等地防疫管控也全面升级，衢州花园小区业委会成员还临时当起“代办员”，通过微信收集业主的采购信息，帮业主“跑腿”，并送货上门。
业委会主任杨利庆：我们觉得应该把小区限制出门的人数越少越好，所以想出来能够尽量满足业主这些基本生活要求，给他们去采购点菜。
广东省卫生健康委防控领导小组办公室主任周紫霄介绍，目前广东正积极有序引导返程、复工，并采取数字化的手段，一键办理信息登记，快速通关。同时，要求广东省内各级医疗机构做好院内患者的流行病学调查工作。
周紫霄：在入粤通道设置联合检疫站，对所有来往人员进行体温检测。一线的医护人员要对门诊、急诊、住院的患者多问一句，是不是来自湖北？凡是14天（1月16日后），有湖北旅行或者是居住史的患者，我们进行一个必查。
湖北省北部紧邻河南信阳，信阳市对所有小区和城中村实行网格化管理、地毯式排查，信阳高速交警支队新区大队实行24小时不间断轮流上岗执勤，对过往车辆实行逢车必查，逢人必检。
沪陕高速信阳新区站点位负责人刘宏杰：发现确有高温的，按照程序移交专业处置部门进行隔离处置诊治。对重点疫区车辆及人员，进行有效劝返，做好沪陕高速道路运输畅通和应急运输绿色通道保障工作。

>>>【央视快评】毫不放松“两手抓” 坚决夺取“双胜利”

“确保打赢疫情防控的人民战争、总体战、阻击战，确保实现决胜全面建成小康社会、决战脱贫攻坚目标任务。”近日，习近平总书记在中共中央政治局常务委员会会议上发表重要讲话，再次强调确保实现疫情防控和经济发展“双胜利”的目标任务。总书记的重要讲话是指导我们准确把握经济社会发展态势的科学判断，充分彰显了以人民为中心的发展理念。各级领导干部要坚定信心、再接再厉，在“两手抓”中努力奋力夺取“双胜利”。

>>>安庆治愈者：庆幸得到了及时有效的治疗

2月6日，是严瑜 (化名)出院回家的第三天，他专注地坐在电脑前，不停地点击着鼠标，剪辑着生病之前拍的视频，他是一名摄 影师，也是安庆市首批三名治愈出院的新型冠状病毒肺炎患者之一。
<span style="'yes';;
;">“我觉得自己就是生了一场病，比平时感冒要严重一些，庆幸的是得到了及时有效的治疗。”谈起自己被感染、被救治，最终康复出院，严瑜都显得很淡然。
两赴武汉
他高烧咳嗽全身无力
<span style="'yes';;
;">严瑜是一名“90后”，前不久刚满26岁。2015年，严瑜从武汉一所大学毕业，几经漂泊，他最终选择回到安庆老家创业。从此，武汉便成为他“总想回去看看”的地方。
<span style="'yes';;
;">1月9日，因为“五年一小聚，十年一大聚”的同学之约，又逢“最要好的大学室友小孩过五周岁生日”，严瑜前往武汉，这一天距离1月12日世界卫生组织正式将造成武汉肺炎疫情的新型冠状病毒命名为“2019新型冠状病毒”还有三天。
<span style="'yes';;
;">当天在武汉相聚的只有三个人，包括严瑜、身为武汉本地人的室友及一名家住湖北省另一座城市的同学。室友对严瑜的到来很是欢迎，专门收拾了一个房间给他住。三个人在武汉“逛吃逛吃”，谁也没有因为新闻里报道的“不明原因肺炎病人”而焦虑。
<span style="'yes';;
;">因家里有事，1月12日，严瑜返回老家，至此，严瑜的身体状况都很正常。本来，严瑜的故事可以在此画上一个句号。但让人没想到的是，三天后，他接到了一个拍摄单子，目的地是武汉。
<span style="'yes';;
;">依旧没有考虑太多，年轻的严瑜扛着相机再次出发武汉，顺利完成拍摄任务后，1月17日左右，他开始觉有自己有些不舒服。“有一点咳嗽，头也昏昏的，浑身没劲，我当时只是觉得自己感冒了。”
<span style="'yes';;
;">感冒了，先到药房买药吃!这是严瑜从小到大都坚信的“真理”。他就近在药房买了一些消炎药，“店员好像还给我拿了莲花清瘟胶囊，但我吃了两天药并没有好转，反而加重了。”严瑜回忆说，体温最高达到38.8℃，咳嗽厉害，全身没力气。
<span style="'yes';;
;">严瑜再次前往药房买药，这次他被拒绝了，对方让他去医院做检查。直到此时，严瑜才开始有点“紧张”。他立即前往武汉一家大型医院就诊，医生很快给他开了肺部CT和核酸检测。那一刻，他还并不知道核酸检测就是确诊“新型冠状病毒肺炎”的重要依据。
<span style="'yes';;
;">检查结果很快就出来了，CT显示“两肺炎症”，但核酸检测结果是阴性，这让严瑜松了一口气，“医生建议我留院观察，但他又说没有床位，开了一些药让我回去吃，嘱咐我可以去中小型医院再看看。”
<span style="'yes';;
;">第二天，腊月二十六，距离年三十还有四天，严瑜去了另外一家医院，医生给他开了药，要在门诊输液。“门诊输液的人很多，我拿了药，要排好久的队才能吊上水。”
对症治疗
四天后病情明显缓解
<span style="'yes';;
;">连输了三天液，严瑜的病情时好时坏。1月22日，腊月二十八，孤身在外的严瑜想回家了。第二天凌晨一点，他坐上“滴滴打车”往桐城赶。在路上，他联系了自己的父亲，并嘱咐他”你一个人到医院等我，其他人都不要过来。”
<span style="'yes';;
;">严瑜直接去了桐城市人民医院，说明情况后，他立即被安排住进了隔离病房，重新拍了CT，抽了血。“他们再次采集了我的痰液，说是送到安庆市里检测。”当天晚上八点多，检测结果反馈到医院，是阳性。
<span style="'yes';;
;">“之前我一个人在武汉还是有一些担心的，但回到家乡，看到有这么多医务人员在，我心里真的就不怕不慌了。”
<span style="'yes';;
;">很快，严瑜被送往安庆市立医院北院区，此时已是大年三十的凌晨。严瑜的父亲也被送到另外一家医院隔离观察。到了安庆市立医院，做了一系列的检查后，严瑜躺在病床上输液。“当时那种感觉很奇妙，生病了应该是不开心的，但当我躺在病床上，看着医生护士围着我，心里很安定。”
<span style="'yes';;
;">“严瑜算是普通型患者中比较严重的，最高发热到39.2℃，肺部病灶范围比较广，有胸闷、气喘、食欲差的症状。”严瑜的主治医生，安庆市立医院北院区感染性疾病科副主任医师刘贞君告诉记者。
<span style="'yes';;
;">在对严瑜的救治过程中，刘贞君除了严格按照国家发布的诊疗指南外，还邀请专家组对其进行会诊，开展个体差异化的“对症治疗”，这其中还包括省里的中医药专家，针对严瑜存在的“舌苔厚腻”等症状进行了中成药治疗，“对严瑜的治疗包括输液、吃药、雾化、吸氧……整个治疗过程都非常顺畅，见效也很快，四天后，严瑜的病情基本上就正常了。”
<span style="'yes';;
;">临床症状缓解、三天体温正常、肺部CT正常、两次(隔一天)核酸检测呈阴性……2月4日，经过各项检查符合治愈出院标准后，严瑜走出病房，成为安庆市首批治愈出院的新型冠状病毒肺炎病人之一。
<span style="'yes';;
;">“医护人员都非常认真负责，我有任何不舒服，或者其他任何问题，他们都会及时帮我处理，并且经常过来陪我聊天，缓解我的紧张情绪。”严瑜告诉记者，每天三餐也吃得都很好，品种很丰富，还有水果。
期盼回归
尽快过上正常生活
<span style="'yes';;
;">在记者采访严瑜的一个小时里，他多次提到，“人人都会生病，我只不过是得了一场名叫新冠肺炎的病。”严瑜面对疾病能够淡定，问及原因，他说“首先，这个病并没有让我完全不能行动，只是身体上有些不舒服;其次，医院的医务人员在积极的治疗我，我也非常信任他们;而且，我每天都可以和家里人视频聊天，互报平安。”
<span style="'yes';;
;">目前，严瑜正在家中继续隔离观察，他的父亲也于2月5日下午解除隔离出院了。“父亲的情况一直正常，家里人都挺好的，我希望能尽快度过隔离期，尽快回到正常工作状态。”（白萍 张冰）

连日来，中国抗击新冠肺炎疫情的有力举措和积极成效受到海外媒体高度关注。许多媒体刊发报道和评论指出，中国防控疫情高效有序展开，充分彰显了中国共产党的领导和中国特色社会主义制度的显著优势。
“防控措施取得积极成效的原因，在于中国共产党的领导和中国体制优势”
多家海外媒体在报道中指出，中国为应对疫情采取的措施快速、高效、有力。德国《世界报》网站刊文指出：“面对紧急形势，中国制度发挥出巨大效力。强大的应对措施只有在人力物力充足和管理完善的制度下才可能实现。”
俄罗斯发行量最大的报纸《劳动报》发表文章认为，确诊病例数在中国不断下降，这无疑要归功于中国政府采取的强有力举措。“关键时刻，更见中国制度优势。”从中央到地方，中国展现出很强的动员能力和组织能力，能在紧急情况下把资源分配到最需要的地方。“正是中国的体制优势，有效遏制了疫情蔓延。”
《俄罗斯报》发表俄专家文章称：“中国政府正在采取严格坚决的措施，如此大规模的社会动员在世界公共卫生史上前所未有。防控措施取得积极成效的原因，在于中国共产党的领导和中国体制优势。”

“中国速度和中国力量向世界展示了抗击疫情的必胜信念和积极成效”

>>>探访治愈出院患者隔离点：“为安全回家做最后准备”

(抗击新冠肺炎)探访治愈出院患者隔离点：“为安全回家做最后准备”
中新社武汉2月29日电 题：探访治愈出院患者隔离点：“为安全回家做最后准备”
中新社记者 杨程晨
29日中午，作为康复期患者隔离点的武汉软件工程职业学院5栋宿舍楼响起了音乐，人们戴上口罩走出房门，站在“口”字型走廊上练起八段锦和太极拳。

图为治愈出院患者在练八段锦。 安源 摄
武汉市新冠肺炎疫情防控指挥部日前发出的第16号通告，要求对治愈出院患者实施14天康复隔离及医学观察。该市相关指定场所近期开始大量收治病患。
武汉软件工程职业学院自21日起有出院病患入住，截至28日共收治了650余人。中新社记者29日获准进入这一隔离点，与战胜疾病、亟待回家的病患零距离对话。
这一由学生宿舍改建的定点，保证患者一人一间房单独隔离。校方为每位病患提供日常必需品及餐食，患者初期入住可根据三张生活指南了解各项须知。

医护人员为治愈出院患者测血压。 安源 摄
“我们每天都会自行测量体温并上报，医生一天至少查房一次，(隔离点)还为我们配备中药和灸贴。”26日入住的王姓教育工作者对记者说道。
中国工程院院士钟南山27日判断，感染后的患者应该不会再被感染；是否会再传染给别人还有待观察。为此，武汉软件工程职业学院副校长孙美华指出，学校制定了严格措施以保障隔离点内外安全；至今未出现出院患者再次传染人的情况。

治愈出院患者在隔离点上网课。 安源 摄
该学院占地1000余亩，校内目前无学生留住。从校门行至隔离宿舍，学校设置了多道“关卡”。“坐镇”指挥部的武昌区副区长向悦介绍，隔离点由区政府、武汉软件工程职业学院、武汉工控工业技术研究院及多家医院等多部门联合指挥。
她特别提到，指挥部为每栋宿舍提供24小时不间断网络客服，武汉工控工业技术研究院专门为此开发了一套系统，并提供人员支持。24日，中国科学院院士仝小林和中国工程院院士董家鸿先后来到该隔离点探访。当前，董院士团队正与技术部门对接，计划推出远程诊疗平台帮助患者在此能被同时监测、治疗其他基础疾病。

医护人员在梳理治愈出院患者资料。 安源 摄
该定点目前对于康复期患者的治疗以中医药为主。湖北省中医院医务处副处长张清称，包括看诊、用药、膳食等在内，隔离点为病患提供了详细的康复或治疗期方案。
他表示，患者能够出院说明其身体症状得到了一定程度的改善，14天的隔离是为了在此基础上让他们的肺功能进行恢复，使免疫力得到进一步提升。这些患者经过观察期，对社会、对家庭都是相对安全的。

治愈出院患者在练八段锦。 安源 摄
据介绍，21日至23日最早一批进入观察期的百余患者将经过多项检测达标后，于3月7日回家。
“我是很愿意被隔离的。”64岁的吴先生被告知出院时武汉尚未出台相关通告，他感到高兴的同时也开始发愁：“我该去哪儿？跟家人在一起对他们不太好，单独居住生活又得不到照顾。”
出院前两日，吴先生得知了14天康复隔离的消息，“这正好帮我解决了难题。”他说，“我要在这里为安全回家做最后的准备。”

治愈出院患者展示自己创作的漫画。 安源 摄
“我没想到是以这样的方式重回校园。学生宿舍各种设施条件都还比较完备，服务也到位。”兼职插画师的黎女士把过去大半个月由患病到康复的经历画成了一幅幅漫画。她将这些作品上传社交网络，获得了大批网友的鼓励。
黎女士表示，从家中转至方舱医院治疗、康复后再前往隔离点观察隔离，“这一路走来挺感慨的”。对于各项指标合格之后便可同家人团聚，她感言，当下心情倍感雀跃。(完)
